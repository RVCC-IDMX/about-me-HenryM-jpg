The background-image property in CSS is used to add one or more images to the background of an element. The image can be set by using background-image: url(). A similar effect can be achieved by using background: url(), but this will overwrite the background color. When setting multiple background images they will be layered on top of each other with the first url() given as the top layer and the next on the layer beneath the first, the next on the layer beneath that one and so on.

The background-repeat property in CSS is used to control how the background images will repeat. By default the image will act the same as if it was set to background-repeat: repeat;. Which will fill the background with the image across the x and y axis. Background-repeat: unset; will essentially function identical to the former. Repeat-y will repeat the image only across the y axis. Repeat-x will repeat the image only across the x axis. If two values are set such as background-repeat: repeat repeat; (the default) the first will apply to the x axis and the second will apply to the y axis. The round property will stretch the image until there is room for another, leaving no gaps; this will affect aspect ratio. The space property distributes whitespace evenly between the elements to avoid clipping.

The background-size property in CSS is used to set the size of the background image of the element. A width and height value can be given such as background-size: 100px 200px;. If a width value is given but no height value is specified the height value will default to auto. The contain value will cause the image to scale as large as possible without stretching or clipping. If the container is larger than the image then the image will repeat unless the background-repeat property is set to no-repeat. The cover value will set the image to the smallest possible size where a single image will cover the entire container. The image will be cropped if it is a different size than the container.

The background-position property in CSS will set the position of the background image. The keywords left, right, center, top, bottom, and center can be set to move the image to the left, right, center, top, bottom, and center of the element. The exact position of the image can be specified using a one value, two value, three value, or four value syntax which will include the x position, y position, keywords, and offset value.

The background-origin property in CSS will set the backgroundâ€™s origin from the start of the border, inside the border, or inside the padding. The border-box value will position the image relative to the border box. The padding-box value will position the image relative to the padding-box. The content-box value will position the image relative to the content box.

The background-clip property in CSS will set whether the background will set how far the background extends. The border-box value will extend the background to the outside edge of the border. The padding-box value causes the background to extend to the outside edge of the padding. The content-box option will clip the background to the content box. The text value will clip the background to the foreground text.

Below is an example of these CSS properties:

#example{
padding: 10rem;
background-image: url(/img/640w.jpg);
background-repeat: repeat;
background-size: contain;
background-position: center;
background-origin: border-box;
background-clip: border-box;
}
